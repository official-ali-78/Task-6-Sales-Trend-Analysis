# 📊 Sales Trend Analysis Using SQL

## 📌 Project Overview

This project analyzes monthly sales trends using SQL aggregate functions. The objective is to calculate monthly revenue, order volume, and identify top-performing months.

---

## 🎯 Objectives

- Analyze monthly revenue
- Analyze monthly order volume
- Group sales by year and month
- Find top-performing months
- Practice SQL aggregate functions

---
